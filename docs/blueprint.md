# **App Name**: MechMind

## Core Features:

- Automated Status Updates via WhatsApp: Automatically send updates to customers regarding their service status and appointment reminders, integrating with either Twilio or Meta's WhatsApp Cloud API.
- AI-Powered Diagnostic Assistance Tool: Provide mechanics with an AI tool that assists in diagnosing vehicle issues based on reported symptoms and vehicle history.
- Order Summary Generation with AI: Automatically generate summaries of service orders using AI to capture essential details like services performed, parts replaced, and total cost, supporting technician record-keeping.
- Vehicle History Analysis Tool: Enable the AI to analyze the service history of a vehicle to predict potential issues, recommend maintenance, and personalize service offerings. It will be a tool used in diagnostic and customer service processes.
- Subscription Management: Implement a subscription management system to control feature access based on the selected plan (PRO, PRO+, PREMIUM).
- User Roles and Authentication: Implement user authentication and authorization using JWT.  Establish user roles such as ADMIN and OFICINA to manage permissions within the application.

## Style Guidelines:

- Primary color: Deep blue (#2962FF), evokes trust, reliability, and technological expertise.
- Background color: Light gray (#F0F4F8), to ensure legibility, with minimal contrast to the other design elements.
- Accent color: Vibrant purple (#9D33EA) to indicate calls to action and key interactive elements within the user interface.
- Body and headline font: 'PT Sans', a humanist sans-serif that's both modern and welcoming, for a balance of technical and approachable content.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use detailed, informative icons with an outline style to convey specific functions related to mechanics, diagnostics, and status updates.
- Employ a mobile-first design with a clean, card-based layout that scales across devices to maintain visual clarity and ease of navigation.
- Use subtle animations to enhance the user experience during key interactions, like updating service statuses, loading diagnostic results, or transitioning between sections.
